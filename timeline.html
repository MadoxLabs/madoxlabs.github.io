Timeline Tool
Create new line : click left edge
Change line default color : click the line thumb to change, click on a new color
Change the line default text : click on the line's circle to use editor
Create new chapter mark : click top edge
Move chapter mark : drag chapter marker left/right
Delete chapter mark : drag chapter marker down
Change marker to push mode : click marker thumb
Change the marker default text : click on the marker's circle to use editor
Create new event : click on a line
Move event : drag event left/right
Resize event : drag right edge of event left/right
Move event to new line : drag event up/down to other line
Delete event : drag event up/down off of line
Change event color : click the event to change, click on a new color
Change the event text : click on the event's text to use editor
Scroll view : drag the black area left/right
Zoom View : drag the black area up/down
