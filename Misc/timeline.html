Keyboard & Mouse Reference

Lines (Character Tracks)

Click left edge to create
Click thumb to change color
Click circle to edit name

Chapters

Click top header to create
Drag left/right to move
Drag down to delete
Click thumb to toggle push mode
Click circle to edit text
Click ■ top-left for chapter reorder

Scenes

Click second header row to create
Drag left/right to move
Drag down to delete
Click thumb to toggle push mode
Click circle to edit text
Click ■ for scene reorder

Events

Click on a line to create
Drag left/right to move
Drag right edge to resize
Drag up/down off line to delete
Click event body to change color
Click event text to edit

Navigation

Drag canvas left/right to scroll
Drag canvas up/down to zoom
